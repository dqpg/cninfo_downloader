# CNinfo-Local-Sync (巨潮研报/公告全自动同步工具)

本项目是一款专为**金融投资从业者**和**个人投资者**设计的本地数据抓取工具。它可以全自动、结构化地同步 A 股及港股的定期报告（年报、季报等）和重要动态公告，为构建本地 AI 研报知识库（如配合 AnythingLLM + DeepSeek）提供高质量的数据源。

> **致谢**：本项目受开源项目 `jarodise/CNinfo2Notebookllm` 启发并在其基础上进行重构，针对本地化 RAG 场景进行了深度优化。

## ✨ 核心亮点
- **傻瓜式交互**：无需记忆复杂的命令行参数，运行后直接进入“一键式”向导。
- **智能时间轴逻辑**：自动对齐 $y-5$ 历史长周期，远期只看年报，近期全量抓取。
- **RAG 友好命名**：文件名包含代码、简称、财报季度（Q1-Q4），让大模型检索更精准。
- **自动排雷监控**：智能捕获问询函、计提减值、股权激励等对股价有重大影响的公告。
- **修订版自动追踪**：自动识别修订版报告并进行标准标注。

## 🚀 极速使用指南
- **方式一：单只股票同步**

双击运行 sync_reports.py，根据提示操作：

输入：输入股票代码（如 300308）或简称（如 中际旭创）。

选择模式：默认 3（财报+重要公告）。

设定年份：直接按回车即使用默认的“近 5 年财报+近 3 年公告”配置。

- **方式二：批量“扫货”模式**

在脚本同目录下新建 pool.txt。

每一行写一个股票代码或名称，如：

贵州茅台
002428
宁德时代

运行脚本，在第一个输入框填入 pool.txt 即可。

## 📁 归档逻辑与命名规范

定期报告：代码_名称_年份_Qx_报告.pdf（Q1:一季报, Q2:半年报, Q3:三季报, Q4:年报）。

重要公告：日期_公告标题_类型.pdf（如：监管、业绩、风险、资本）。

修订处理：若存在修订版，文件名将自动追加 _修订版 后缀。
